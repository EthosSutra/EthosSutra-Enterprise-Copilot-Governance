# Control Library

## Purpose

The Control Library establishes the governance, risk management, security and operational controls implemented for the EthosSutra Enterprise Copilot.

The objective is to provide a centralized inventory of controls used to manage AI-related risks and support governance, compliance and audit activities.

The Control Library serves as the primary link between identified risks, implemented controls and supporting evidence.
